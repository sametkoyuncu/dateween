# What is this?
Get the number of days (or seconds, minutes etc.) between two dates in JavaScript.

# Installation

`npm i dateween --save`

# Usage

```
const dateween = require('dateween');

console.log(dateween.getDays("2021-12-3", "2020-1-24"));
```
